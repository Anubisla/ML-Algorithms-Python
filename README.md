## 📌 2. ML Prediction Web App

A simple end-to-end Machine Learning application built using Flask.

### Features
- Takes user input (study hours, sleep hours)
- Predicts student performance
- Uses trained ML model (Linear Regression)
- Model saved using Pickle

### Tech Stack
- Python
- Scikit-learn
- Flask
